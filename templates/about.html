ABOUT SKYSNAP Weather

🌞

Welcome!

SKYSNAP is your friendly weather app that helps you check current weather, forecasts, trends, and historical data worldwide. Get accurate weather insights anytime!

🌟

Features

  • Real-time weather updates for any city
  • 5-day forecast and 3-hourly data
  • Temperature, wind, and precipitation graphs
  • Compare weather between two cities
  • Track historical weather trends
  • Analytics and trend insights
👩‍💻

Developer

Created by Team SKYSNAP, exploring Python, Flask, and web APIs.

📡

Data Sources

  • OpenWeather API — current weather & forecast
  • OpenStreetMap Nominatim API — reverse geocoding
