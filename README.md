# Weather App - Python Flask 🌦️

Real-time weather application with Flask backend and API integration.

**Tech Used:** Python, Flask, HTML5, CSS3, JavaScript, OpenWeatherMap API

**Features:**
- Search weather by city name
- Live temperature, humidity, wind speed  
- Mobile responsive design
- API error handling

**How to Run Locally:**
1. Clone: `git clone https://github.com/PoojarySushmita/weather-app-flask.git`
2. Install: `pip install flask requests`
3. Add your API key in `app.py`
4. Run: `python app.py`
5. Visit: `http://localhost:5000`

**What I Learned:** Flask routing, REST API calls, JSON parsing, Frontend-Backend connection
